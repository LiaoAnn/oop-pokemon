# oop-pokemon
## Introduction
```
1. 先進入文檔(case.txt) 裡面有三個檔案(PokemonLib.txt/ MoveLib.txt/GameData.txt)
2. PokemonLib.txt 為屬性及數值，數值為防禦和攻擊
3. MoveLib.txt為技能名稱、寶可夢屬性和物理或特殊攻擊，後面則是傷害值，若是特殊攻擊會有特殊效果
4. Gamedata.txt 第一行為寶可夢的數量，第二行為寶可夢名稱，第三行為攻擊招式的名稱
```

## How to run

### Docker Compose

```bash
docker-compose up -d
```
